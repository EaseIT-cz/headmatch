# TASK-094: Housekeeping – GitHub issue templates

**Summary**: Add GitHub issue templates for bug reports and feature requests.

**Context**: The repository currently lacks issue templates, leading to inconsistent reporting.

**Scope**:
- Create `.github/ISSUE_TEMPLATE/bug_report.md` and `.github/ISSUE_TEMPLATE/feature_request.md` with standard sections.
- Ensure the repository settings enable the templates.

**Out‑of‑Scope**:
- Custom templates for other issue types.
- Automation of issue triage.

**Acceptance Criteria**:
- Both template files exist in the repo.
- New issues default to the appropriate template.
- Templates include placeholders for description, steps to reproduce (bug), and motivation (feature).

**Suggested Files/Components**:
- `.github/ISSUE_TEMPLATE/bug_report.md`
- `.github/ISSUE_TEMPLATE/feature_request.md`
