# HeadMatch 0.5.1

## Summary

Adds APO preset refinement (CLI + GUI), broadens CI to Python 3.10–3.13, and hardens PipeWire capture.

## Features

- **APO refine mode**: `headmatch refine-apo --preset eq.txt --recording rec.wav --out-dir refined/` loads an existing Equalizer APO parametric preset and re-optimises its bands against a fresh measurement using joint Nelder-Mead refinement. The fit report includes before/after RMS error comparison. GUI: the Import APO screen now includes a "Refine against a measurement" section with recording WAV picker, optional target CSV, and output folder.
- **CI coverage reporting**: New `coverage.yml` workflow runs `pytest-cov` on main pushes and uploads `coverage.xml` as a build artifact.

## CI

- **Python 3.10–3.13 matrix**: Unit tests run across all four versions. Integration tests run on 3.10 and 3.13 bookends.

## Reliability

- **PipeWire capture pipe hardening**: `pw-record` stdout redirected to DEVNULL (prevents pipe-buffer deadlock on long recordings). stderr persisted to `pw-record-stderr.log` in the output directory for post-mortem diagnosis.

## Removed

- **`fit-offline` CLI alias**: Removed entirely. Was identical to `fit` with no reported usage.

## Tests

432 → 436 deterministic tests (+4 refine mode tests).

## Upgrade

```bash
pip install --upgrade headmatch
headmatch --version

```
