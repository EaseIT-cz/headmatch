# HeadMatch 0.4.5

## Summary

Fixes target editor CSV loading to preserve curve shape.

## Fix

- **Target editor from_csv**: Was hardcoded to sample exactly 8 points regardless of input size. A 6-point CSV lost data; a 479-point measurement grid lost all shape detail. Now loads all points directly for small CSVs (≤24 points), or intelligently downsamples to ~24 control points by picking peaks, troughs, endpoints, and evenly-spaced fill samples for dense grids.

## Upgrade

```bash
pip install --upgrade headmatch
headmatch --version
# headmatch 0.4.5
```
