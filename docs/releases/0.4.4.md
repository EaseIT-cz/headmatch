# HeadMatch 0.4.4

## Summary

Bug fix, UX improvements, and desktop shortcut management.

## Bug fix

- **Dense GraphicEQ clipping**: The GraphicEQ export was writing the raw correction target at full 48 PPO resolution, including extreme swings at narrow features the PEQ fitter deliberately ignores. Now exports the actual PEQ-fitted response, matching what the parametric preset applies.

## UX improvements

- **Target editor: Load CSV**: New "Load CSV" button loads an existing target curve into the editor for modification.
- **Target editor: per-row add**: Each control point row has a "+" button that inserts a new point between it and the next, at the midpoint frequency. Replaces the confusing single "Add point" button.
- **History: Browse button**: Added "Browse…" button next to the search folder field.
- **Desktop shortcut management**:
  - `headmatch create-shortcut` / `headmatch remove-shortcut` CLI commands
  - `headmatch doctor` suggests creating a shortcut when GUI is found but not installed
  - GUI Setup Check has a toggle button to create/remove the desktop launcher

## Test suite
392 tests, all passing.
