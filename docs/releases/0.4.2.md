# HeadMatch 0.4.2

## Summary

Fixes the target curve editor to be fully interactive.

## Changes

- Target editor control points are now editable with text fields and gain sliders (-20 to +20 dB).
- Add and remove point buttons work with immediate re-render.
- Apply changes reads all fields back into the editor model before save.
- Minimum 2 control points enforced.

## Test suite
392 tests, all passing.
