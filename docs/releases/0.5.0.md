# HeadMatch 0.5.0

## Summary

Major feature release: real headphone database search, live target editor preview, performance overhaul from external code review findings, and significant internal architecture improvements.

## Features

- **Headphone database search**: `headmatch search-headphone "HD 650"` queries the AutoEQ GitHub repository via API, returns matching models with source and form factor metadata, and prints copy-paste `fetch-curve` commands. Results cached locally for 24 hours. GUI includes a search field in the Fetch Curve view with selectable results.
- **Live curve preview in target editor**: Canvas widget renders the PCHIP-interpolated target curve in real time with log-frequency X axis (20–20 kHz), dB grid (±20 dB), octave boundary lines, and draggable control point markers. Updates on every Apply click.

## Performance

- **O(N) fractional-octave smoothing**: Replaced the O(N²) NxN Gaussian weight matrix with log-domain resampling + `scipy.ndimage.gaussian_filter1d`. Output matches the old implementation within 0.001 dB on the default grid. Dense 10k-point curves complete in ~4 ms instead of quadratic blowup.
- **C-backed alignment peak detection**: Replaced Python for-loop local maxima scan with `scipy.signal.find_peaks` in the cross-correlation alignment step.

## Architecture

- **Explicit shelf parameter semantics**: Added `PEQBand.slope` field to distinguish shelf slope S from peaking Q. New `effective_slope` and `shelf_q` properties. Fully backward compatible — existing code that stores slope in the `q` field still works via automatic fallback.
- **Pipeline module split**: Extracted `pipeline_confidence.py` (scoring algorithm + threshold constants) and `pipeline_artifacts.py` (export writing + README generation) from `pipeline.py`, reducing it from 547 to 267 lines. Public API unchanged.

## Fixes

- UTF-8 encoding specified on all file I/O across 11 modules (prevents locale-dependent failures on Windows).
- `fetch_curve_from_url` raises `ValueError` on non-UTF-8 downloads instead of raw `UnicodeDecodeError`.
- Cache directory falls back to temp dir when home directory is read-only.

## Packaging

- Added `pytest.ini` with `pythonpath` config — tests runnable without editable install.
- Added `MANIFEST.in` for complete sdist (LICENSE, docs, tests, changelog).
- Added `[tool.setuptools.packages.find]` to exclude `tests/` from wheel.

## Tests

401 → 432 deterministic tests (+31). New coverage: smoothing regression and scalability, shelf semantics round-trip, alignment robustness (delay + echo + noise), headphone DB search/caching/network fallback, CSV parsing, curve preview rendering.
